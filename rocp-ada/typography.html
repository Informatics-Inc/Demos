Typography Guide: Words That Work

Design is how it works and how it reads. This page is your typography playground—where spacing, scale, rhythm, and hierarchy turn plain text into clear communication.

Our Mantra

Clarity over cleverness. Good type invites the eye, guides the mind, and never shouts without purpose. Make it readable, then make it delightful.

“Great design is invisible until you need it.” —Someone who kerns their cereal boxes

Headings (H1–H5)

H1 · Big Idea, Single Focus

Use for page titles and major section intros. Keep it short; let the content breathe.

H2 · Section Heading

H2s group related content. If you need a quick scan, these are the highway signs.

H3 · Subsection Heading

Helpful for breaking down complex topics. Think chapters within chapters.

H4 · Supporting Heading

Use for labels, component titles, or small feature callouts.

H5 · Fine Print Heading

When you need structure without shouting—H5 gets the nod.

Paragraphs, Emphasis, and Inline Elements

Words carry weight. Use strong to emphasize importance, emphasis to change tone, and e.g. to keep things tight. Save ALL CAPS for labels, not paragraphs. Don’t forget semantic helpers like <abbr>, <mark>, <sup>/<sub>, and <kbd>.

Shortcut of the day: Press Ctrl + S (or Cmd + S) to save, then sip water. Hydration and typography both prevent headaches.

Chemical vibes: H2O is refreshing; E = mc2 is iconic.

Lists That Actually Help

Unordered List

  • Readable line length (45–75 characters)
  • Comfortable line height (at least 1.4)
  • Consistent spacing above and below blocks

Ordered List

  1. Set your base size and scale
  2. Establish heading hierarchy
  3. Tighten contrast and color
  4. Ship, learn, refine

Description List

Kerning
Adjusting space between specific letter pairs for harmony.
Leading
Line spacing. Give paragraphs room to breathe.
Tracking
Uniform spacing across a range of text. Use lightly.
